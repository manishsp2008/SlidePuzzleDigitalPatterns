# Tech Stack

## Framework & Runtime
- **Application Framework:** Next.js (configured for Static Site Generation `output: 'export'`)
- **Language/Runtime:** TypeScript / Node.js
- **Package Manager:** npm

## Frontend
- **JavaScript Framework:** React
- **CSS Framework:** Tailwind CSS
- **UI Components:** Lucide React (for icons), standard HTML/CSS elements for the grid

## Testing & Quality
- **Test Framework:** Jest + React Testing Library
- **Linting/Formatting:** ESLint, Prettier

## Deployment & Infrastructure
- **Hosting:** Static Hosting (e.g., GitHub Pages, Vercel, Netlify)
- **CI/CD:** GitHub Actions (standard for static site build/deploy)
