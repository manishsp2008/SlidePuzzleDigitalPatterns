# Core Pattern Generator & UI

## Description
A digital companion web application that generates infinite pattern challenges for a physical slide puzzle toy. 

## Key Elements
- **Pattern Generation Logic:** Create valid, solvable arrangements of 4 colors (Red, Yellow, Green, Blue).
- **Visual Display:** Render the generated pattern clearly for a child to replicate on their physical board.
- **Difficulty Settings:** Allow users to choose between difficulty levels (e.g., Easy, Medium, Hard) which affects the complexity of the generated pattern.
- **Controls:** "New Card" button to generate a fresh pattern.
